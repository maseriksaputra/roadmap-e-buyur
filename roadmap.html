‚ö†Ô∏èManajemen Risiko & Mitigasi

üé• Kamera Web butuh HTTPS
Webcam tidak berjalan tanpa HTTPS.
Mitigasi: Prioritaskan Android fisik untuk demo; gunakan staging HTTPS/ngrok untuk uji web.
üåê Latensi Mobile ‚Üí Cloud
Packet loss/latensi memengaruhi UX.
Mitigasi: Cache katalog; retry local queue; pagination & debounce.
üîê Kebocoran Role/Policy
Akses seller ke resource bukan miliknya.
Mitigasi: Unit test policy; middleware ownership; audit trail.
üì± Kinerja ML di Perangkat Low-end
Inferensi lambat/boros memori.
Mitigasi: TFLite FP16 + XNNPACK; resize input; batasi resolusi kamera.
üç™ CORS/Sanctum Cookie
Login web gagal karena konfigurasi domain/stateful yang tidak konsisten.
Mitigasi: Pastikan SANCTUM_STATEFUL_DOMAINS & SESSION_DOMAIN konsisten; fallback token untuk mobile.
